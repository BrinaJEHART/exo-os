# eXo - OS

Simple all in one app idea (os like) Calculator, WeatherApp, ImageEditor..... Without a proper file system

## Getting Started

> Download the project
> npm install
> that's it... now just run "npm run start"

### Prerequisites

* NodeJS

## Built With

* electronJS
* jQuery
* Pug
* some random js libraries
